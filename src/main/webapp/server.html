Комплексные авиационные тренажеры предназначены для тренировки и
профессиональной проверки летного состава.Процесс тренировки
определяется программами, которые включают:
• основные сведения о сроках, местах и условиях тренировки;
• перечень упражнений для выполнения экипажем;
• критерии оценки знаний и навыков;
• правила оформления документации тренировки;
• описание процедур проведения упражнений.
Описание процедур проведения упражнений включает:
• условия полета;
• основную схему упражнения (например, взлет, набор высоты, полет в зону);
• отказы и ситуации, отрабатываемые в упражнении;
• временной регламент упражнения.
Программа не формализует конкретные показатели и
значения отдельных оценок знаний и навыков,
ограничиваясь общими указаниями по проведению оценивания и
правилами формирования общей оценки для пилотов.
